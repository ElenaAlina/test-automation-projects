# TEST PLAN - End-to-End automated test suites + BASIC Security
Application: SauceDemo
URL: https://www.saucedemo.com
Tool: Cypress

# Test Objective
The scope is to validate the main functionalities of the application:
 - authentication
 - product management
 - cart
 - business logic - E2E flow

 # Test Scope
 > In Scope
    - Login / Logout
    - Products page
    - Cart functionality
    Basic security tests:
    - input validation
    - simple injection attemps
    - flow validation
> Out of Scope
    - Performance
    - Load testing
    - Advanced security checks

# Test Strategy
> Testing types:
    - End-to-End Testing (E2E)
    - Functional Testing
    - Negative testing
    - Basic Security Testing
> Automation:
    - Page Object Model
    - Fixtures (test data)
    - Custom commands

# Test Environment
    - Browser: Chrome (also can be used with another installed browser)
    - OS: Windows
    - URL: https://www.saucedemo.com
    - Tool: Cypress
