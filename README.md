# Test Automation projects

Cypress-based test automation portfolio covering multiple testing types such as End-to-End (E2E) and basic security scenarios.

## Project Overview

This project demonstrates automated testing practices using Cypress, including:

* Test planning and scope definition
* Test cases and step-by-step scenarios
* UI and functional validation
* Basic security checks

## Test Coverage

The project includes:

* Authentication (login/logout)
* Product interactions (sorting, viewing, adding to cart)
* Cart functionality
* Security-related test scenarios

## Project Structure

* **Test Suites** – organized test cases for different features
* **Page Object Model (POM)** – reusable and maintainable structure
* **Fixtures** – test data management
* **Custom Commands** – reusable Cypress commands

## Reporting

* Mochawesome HTML reports generated after test execution
* Includes test results, logs, and execution details

## How to Run the Tests

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npm test
```

## Technologies Used

* Cypress
* JavaScript
* Mochawesome Reporter
